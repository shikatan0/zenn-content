# 目的

MDN に記述されていなかった機能の紹介。

# 文字列の分割

```js
const [head, ...rest] = 'abc'

head  // 'a'
rest  // ['b', 'c']
```

# 配列の添字で指定

```js
const {0: a, 2: b} = ['00', '01', '02']

a  // '00'
b  // '02'
```
