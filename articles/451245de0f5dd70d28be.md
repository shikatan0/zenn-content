# Windows.Media.Ocr

`Windows.Media.Ocr` は画像からテキストを読み取る機能、光学文字認識 (OCR: Optical character recognition) を行うための Windows Runtime API です。

https://docs.microsoft.com/en-us/uwp/api/windows.media.ocr

# 環境

|対象|バージョン|
|:-:|:-:|
|Windows|20H2 (19042.928)|
|.NET SDK|5.0.202|

# 実装例

指定した画像ファイルからテキストを読み取り表示するサンプルです。

```xml:.fsproj
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>net5.0-windows10.0.19041.0</TargetFramework>
    <RootNamespace>プロジェクト名</RootNamespace>
  </PropertyGroup>

  <ItemGroup>
    <Compile Include="Program.fs" />
  </ItemGroup>

</Project>
```

```fsharp:Program.fs
open System

open Windows.Foundation

let await (i: IAsyncOperation<'TResult>) =
    i.AsTask() |> Async.AwaitTask

open Windows.Media.Ocr
open Windows.Storage
open Windows.Graphics.Imaging

let printOcrResult () =
    let ocrEngine = OcrEngine.TryCreateFromUserProfileLanguages()

    async {
        let! file      = await <| StorageFile.GetFileFromPathAsync @"画像ファイルへの絶対パス"
        let! stream    = await <| file.OpenAsync FileAccessMode.Read
        let! decoder   = await <| BitmapDecoder.CreateAsync stream
        let! bitmap    = await <| decoder.GetSoftwareBitmapAsync()
        let! ocrResult = await <| ocrEngine.RecognizeAsync bitmap

        printfn "%s" ocrResult.Text
    }

printOcrResult() |> Async.RunSynchronously
```

# Windows Runtime API の使用方法について

`.fsproj` ファイル、`TargetFramework` の値は以下から選択可能です。

|Windows 10 Version 1903|net5.0-windows10.0.18362.0|
